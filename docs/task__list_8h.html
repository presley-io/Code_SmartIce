void task_bathsensor ( void *  p_params)

Bath Temperature Sensor Task.

This is the task for the bath temperature sensor. State 0 is initialization, which occurs when the task is first created. State 1 is what runs forever in the for(;;) loop. Since there are only 2 states including initialization, the states were not necessary to define.

This task uses the ADC temperature probe class constructed by the manufacturer of this product.

◆ task_overallcontroller()

void task_overallcontroller ( void *  p_params)

Overal Controller Task.

This is the task for the system controller. This controller employs a PID controller, with only a P gain. State 0 is initialization, which occurs when the task is first created – this defines the controller values as well as initializes the pump objected, which yes belongs to this task. State 1 is what runs forever in the for(;;) loop (besides the if statement that detects a pop). State 2 occurs if a pop is detected.

[C], PID error – difference between desired and measured.

[C], desired temperature at arm

[C], Maximum temp of bath; indicates that water/ice needs to be replenished

[motor_PWM / C], proportional gain of the PID controller.

[kPa], recording of last presure.

[%], percentage pressure drop tolerance. If exceeded, will trigger shutdown.
