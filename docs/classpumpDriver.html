Detailed Description

Pump driver for DC motorized pumps.

This driver uses pwm functionality and logic to control pumps and set limits to their uses. The pump object is assigned to a pin, and its settings can be modified and stored within the object.

void pumpDriver::set_Duty ( float  p_dutycycle)

Functions sets specific duty to a specific pin.

The function simply transports the desired PWM duty to the desired pin. There is some filtering that occurs on the desired duty cycle. This function is useful because we can take the floating point values from our controller algorithm, and send them straight into the motor function.

void pumpDriver::set_max ( float  p_maxsetting)

Sets a maximum pump load.

Adjusts the maximum allowable pump speed. This is useful for pwm applications as the controller can often output speeds that are unnecessary or impossible to hit. The value must be larger than the min_pwm_setting but not larger than 100.
